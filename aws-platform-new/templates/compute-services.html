What are compute services?

AWS Compute services provide the processing power needed to run applications, from traditional virtual servers to serverless functions. These services allow you to choose the right compute option based on your workload requirements.

Main compute services

EC2 (Elastic Compute Cloud)

Virtual servers in the cloud with complete control over your computing resources

Key features:

  • 100+ instance types for different workloads
  • Auto Scaling for automatic capacity adjustment
  • Elastic Load Balancing for high availability

Common use cases:

  • Web application hosting
  • Database servers
  • Development and test environments
Learn more Regional Service

Lambda

Serverless compute service that runs code without provisioning servers

Key features:

  • Pay only for compute time consumed
  • Automatically scales from few requests to thousands
  • Supports Python, Node.js, Java, Go, .NET, Ruby

Common use cases:

  • API backends and microservices
  • Data processing and ETL jobs
  • Event-driven automation
Regional Service

ECS (Elastic Container Service)

Fully managed container orchestration service for Docker containers

Key features:

  • Fargate for serverless containers
  • EC2 launch type for full control
  • Integrated with AWS services

Common use cases:

  • Microservices architecture
  • Batch processing jobs
  • ML model deployment
Regional Service

EKS (Elastic Kubernetes Service)

Managed Kubernetes service for container orchestration

Key features:

  • Fully managed Kubernetes control plane
  • Compatible with standard Kubernetes tools
  • Multi-AZ deployment for high availability

Common use cases:

  • Cloud-native applications
  • Hybrid deployments
  • Machine learning workloads

Best practices

  • Right-size your instances: Choose instance types that match your workload requirements
  • Use Auto Scaling: Automatically adjust capacity based on demand
  • Leverage spot instances: Save up to 90% for fault-tolerant workloads
  • Use reserved instances: Commit to 1-3 years for predictable workloads to save up to 72%
  • Monitor and optimize: Use CloudWatch to track performance and costs
