# Quick Start Commands

## Start the Server

```bash
# Option 1: Using start script (recommended)
./start.sh

# Option 2: Manual
source venv/bin/activate
python app.py
```

## Access the Platform

- **Student View**: http://localhost:5001/
- **Teacher View**: http://localhost:5001/teacher

## Stop the Server

Press `Ctrl+C` in the terminal
